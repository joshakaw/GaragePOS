# Scope of Project

This point of sale system must generally have the same features that are available on the Casio SE-G1 Cash Register. This includes:

- Simple sales
- Refund an item
- Pay In (RA)
- Pay Out (PO)
- Reporting
- PLU tracking
- Tax rate
- Clerk assignment
